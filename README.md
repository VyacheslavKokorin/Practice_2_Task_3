# Practice_2_Task_3
## Требования
Наличие установленного Docker.  
Наличие установленного git для клонирования репозитория.

## Установка и запуск
Перейдите в папку, где будет размещён проект, в терминале. Например:
```
cd ~/Documents
```

Склонируйте репозиторий себе на компьютер
```
git clone https://github.com/VyacheslavKokorin/Practice_2_Task_3.git
```

Перейдите в папку проекта
```
cd Practice_2_Task_3
```

Создайте файл с переменными окружения
```
cp .env_example .env
```
В боевом применении пароли необходимо ОБЯЗАТЕЛЬНО поменять на что-то сложное и уникальное, но в учебном можно оставить так

Запустите контейнеры Docker
```
docker compose up -d
```
